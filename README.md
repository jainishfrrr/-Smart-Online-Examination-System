# Smart Online Examination System

## 🚀 Project Description
This is a full-stack web application developed using JSP, Servlets, JDBC, and MySQL.
It allows users to attempt quizzes, track performance, and provides admin control.

## 🔥 Features
- User Login & Registration
- Session & Cookie Management
- Dynamic Questions from Database
- Timer-based Quiz with Auto Submit
- Anti-Cheating Features
- Admin Panel (Add/Delete Questions)
- Leaderboard (Top Students)
- User Result History
- Reset System (Admin Control)

## 🛠 Technologies Used
- Java (JSP, Servlet)
- JDBC
- MySQL
- HTML, CSS, JavaScript

## 📊 Future Improvements
- Password encryption
- Email verification
- Mobile responsive UI
